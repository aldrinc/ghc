# 01_governance — Foundational Documents

## What This Contains

The immutable source-of-truth documents that govern all copywriting workflows. These are NEVER modified by workflow execution — they are READ-ONLY inputs.

## Structure

### sections/
Core operating documents (Sections 2-11 + Subsections A-E). These define:
- Page-type templates and belief architecture (S2)
- Voice and tone rules (S3)
- Compliance constraints — FDA/FTC/platform (S4)
- Awareness-level routing logic (S5)
- Proof inventory and credibility hierarchy (S6)
- Hook construction psychology (S7)
- A/B testing hypothesis framework (S8)
- Section-level job definitions with CHECK gates (S9)
- Context window management strategy (S10)
- Mental models for self-evaluation (S11)
- Structural principles and belief chains B1-B8 (SA)
- Sentence-level craft rules (SB)
- Historical A/B testing patterns (SD)
- Behavioral science principles for copy (SE)

### shared_context/
Condensed operational extractions loaded by EVERY workflow:
- `brand-voice.md` — Voice positioning, banned/preferred words, emotional register
- `compliance.md` — Platform-specific compliance rules
- `audience-product.md` — **TEMPLATE** — Fill in your brand's specifics
- `mental-models.md` — Self-evaluation protocols

### research_artifacts/
Example research inputs (from The Honest Herbalist project). Replace with your own primary research when starting a new brand.

## Loading Order
1. Load `shared_context/` files first (every workflow)
2. Load relevant `sections/` docs based on workflow needs
3. Reference `research_artifacts/` for audience/market context

## Parameterization
`audience-product.md` uses {PLACEHOLDER} variables. Fill these in for your brand before running any workflow. All other docs are system-level and brand-agnostic.
