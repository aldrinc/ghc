# How to Use These Examples

## The Honest Herbalist — Complete Worked Example

This folder contains a complete end-to-end execution of the DR Copywriting System for a real product: The Honest Herbalist Handbook ($49, laminated spiral-bound herbal reference manual).

### What's Included

**brand_context/** — The filled-in audience-product and brand-voice docs for this specific brand. Use these as a model for how to fill in the template in `01_governance/shared_context/audience-product.md`.

**headlines/** — Multiple rounds of generated and scored headlines across different angles (dosage, dangerous detail) and awareness levels.

**advertorials/** — Two complete presell advertorials (markdown source + Word doc):
- Dosage angle (A6 variant) — scored 17/19 (89.5%) on congruency
- Dangerous Detail angle — scored 17/19 (89.5%) on congruency

**sales_pages/** — Three complete sales page outputs using different template architectures:
- Output A: Section 2 belief-chain template (copy-first)
- Output B: PDP schema JSON (data-first, renderable)
- Output C: Merged optimal (belief chain + UI components)
- Plus: Side-by-side comparison document

**promise_contracts/** — Three Promise Contract JSON examples showing different loop questions and delivery tests.

**test_reports/** — Layer 1 upgrade test report with detailed scoring methodology and findings.

### How to Study These

1. Start with `promise_contracts/` — understand how the Promise Contract governs everything
2. Read an advertorial (markdown) — see how belief chain B1-B4 is built
3. Read a sales page (markdown) — see how B5-B8 continues from the advertorial
4. Open the Sales_Page_Comparison.docx — see how three architectures handle the same brief
5. Check the test report — see the scoring methodology in action
